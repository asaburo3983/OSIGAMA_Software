起動の仕方

　OculusLink（もしくはAirLink）でPCとOculusをつないでPC側で起動してください

  exeファイルはGameフォルダの中の「OSIGAMA.exe」です

　終了はOculus側でメニューからアプリの終了か、PC側でAlt＋F4を押してください
　
操作方法

　Oculusのトリガーボタンでつかんで物を顔まで持っていくだけ

　コーヒーを飲んでスコアをあげつつ、にょとまーるを飲んで
　尿意を止めましょう
